# Type B Library

Store game-specific pre-generated boxes here.

## Recommended structure

type-b/
  <publisher>/
    <game>/

Examples:

- cmon/zombicide-black-plague/
- repos-production/7-wonders-duel/
- stonemaier-games/scythe/

Recommended contents per game:

- box50-<L>x<P>x<H>-typeB.svg
- box50-<L>x<P>x<H>-typeB.dxf
- README.md

The README should document the target game, edition if relevant, supported inserts, sleeve assumptions, and any component-specific constraints.
