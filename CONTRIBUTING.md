# Contributing to Box50

## Rules
- Use Pull Requests
- Reference an Issue
- Follow Box50 dimension rules (50 mm grid, 3mm material)

## Adding a module
- Place files in modules/<name>/
- Include:
  - SVG or DXF files
  - Markdown description
  - Optional preview

## Naming
box-<L>x<P>x<H>-typeA
box-<L>x<P>x<H>-typeB

## Requirements
- Must follow /docs/specification.md
- Must respect tolerances and lid system
