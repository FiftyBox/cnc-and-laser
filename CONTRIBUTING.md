# Contributing to Box50

## Rules
- Use Pull Requests
- Reference an Issue
- Follow Box50 dimension rules (50 mm grid, 3mm material)

## Adding a pre-generated box
- Place Type A files in library/type-a/<L>x<P>x<H>/
- Place Type B files in library/type-b/<publisher>/<game>/
- Include:
  - SVG or DXF files
  - Markdown description
  - Optional preview

## Naming
box50-<L>x<P>x<H>-typeA-layout.svg
box50-<L>x<P>x<H>-typeA-cut.svg
box50-<L>x<P>x<H>-typeB-layout.svg
box50-<L>x<P>x<H>-typeB-cut.svg

## Requirements
- Must follow /docs/specification.md
- Must respect tolerances and lid system
