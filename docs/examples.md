# Examples

## Reference Validation Case — 100×200×100 Type A
Use this box as the primary validation case for the first production SVG phase.

External:
- Width = 100 mm
- Depth = 200 mm
- Height = 100 mm

Internal:
- Width = 94 mm
- Depth = 194 mm
- Height = 90.75 mm

Required first-production panel set:
- bottom
- front
- back
- left
- right
- lid

This case is the baseline for validating nominal dimensions, flat layout, production-only SVG output, and future finger-joint integration.

First joint convention for this validation case:
- front and back: vertical mortise edges
- left and right: vertical tenon edges
- front, back, left, right: bottom mortise edge
- bottom panel: tenon edges on all four sides
- joint pitch: 12.5 mm
- top wall edges remain straight in this phase

First rail convention for this validation case:
- left and right panels contain one rectangular rail slot each
- slot top offset: 3 mm
- slot height: 3.2 mm
- slot length: panel width minus leading material margin minus 3 mm

First Type A mortise-grid convention for this validation case:
- front and back panels contain rectangular mortise slots
- mortise centers: 25 mm, 50 mm, 75 mm
- mortise size: 3.1 mm × 4.5 mm
- mortise bottom edge: 6 mm above the bottom outer edge

First removable divider convention for this validation case:
- one divider template spanning the internal depth
- divider body height: 90.25 mm
- left and right side tenons only
- tenon depth: 2.9 mm
- tenon height: 3.5 mm
- tenon bottom edge: 3 mm above the divider bottom
