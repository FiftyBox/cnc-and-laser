# Tolerances — Box50

Kerf = 0.1 mm

## SVG Cut Compensation
- SVG cut output applies kerf compensation automatically.
- Outer contours are offset outward by kerf / 2.
- Internal cut contours are offset inward by kerf / 2.

## Mortises
- 3.1 mm width
- 4–5 mm height

## Tenons
- 2.9 mm width
- 3–3.5 mm height

## Rails
- 3.1 mm × 3.2 mm

Height = Hint − 0.5 mm
