# Box50 — Technical Specification v1.0

## 1. External Dimensions
Lext = 50 × n
Pext = 50 × m
Hext = 50 × h

## 2. Material
3 mm MDF or plywood.

## 3. Internal Dimensions
Lint = Lext − 6 mm
Pint = Pext − 6 mm
Hint ≈ Hext − 9.3 mm

## 4. Assembly
Rectangular finger joints:
- Tenon: 2.9 mm
- Mortise: 3.1 mm
- Depth: 3 mm

## 5. Sliding Lid
Rail:
- 3.1 mm width
- 3.2 mm height
- 3 mm from top

Lid:
- Width = Lint
- Depth = Pext
- Thickness = 3 mm

## 6. Box Types
### Type A
- Mortises every 25 mm
- Optional bottom grooves
- Removable dividers

### Type B
- No generic mortises
- Fixed dividers only

## 7. Divider Systems
### Type A removable
- Tenons 2.9 mm
- Mortises 3.1 mm
- 25 mm spacing

### Type B fixed
- Finger-jointed mini-panels

## 8. Modules
50×50, 50×100, 100×100, 100×200, etc.

## 9. Tolerances
Kerf = ~0.1 mm
Divider clearance = 0.5 mm
Lid clearance = 0.2–0.3 mm
